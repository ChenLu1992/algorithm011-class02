26. 删除排序数组中的重复项
给定一个排序数组，你需要在 原地 删除重复出现的元素，使得每个元素只出现一次，返回移除后数组的新长度。

不要使用额外的数组空间，你必须在 原地 修改输入数组 并在使用 O(1) 额外空间的条件下完成。


int removeDuplicates(int* nums, int numsSize){
   
   if((0==numsSize) || (null == nums)){
       return 0;
   }

   int i=0;
   
   for(int j=1; j<numsSize; j++){
       if(nums[i] != nums[j]){
           nums[++i]=nums[j];
       }
   }
   
   return i+1;

}
